# Benchwarmer

Warms the page (and backend) caches, using your sitemap.

## Usage

```sh
# Clone repository.
git clone https://github.com/humanmade/benchwarmer.git
cd benchwarmer

# Install dependencies.
composer install

# Run, using a sitemap URL.
php run.php https://example.com/sitemap.xml
```
